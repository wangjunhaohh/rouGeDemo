# v0.3

## 版本目标

把 `v0.2` 文档中的遗留问题和下版本计划真正落地，继续补强可玩性与稳定性。

## 本次新增内容

- 为 Boss 加入专属技能循环，包括扇形弹幕、灼火突进、余烬环爆与召集护卫
- Boss 在血量压到 55% 后进入二阶段，提升移动与弹幕压迫
- 升级项加入联动效果，当前包含过载连射、裂穿扩散、脉冲回流三条组合
- 升级池权重会根据当前 build 倾向动态偏向联动项
- 局外成长从 4 项扩展为 8 项，并加入前置条件分支
- 新增结算碎片增益型局外成长 `残响回收`

## 已修复的问题

- 修复人物走到边缘时相机会看到世界外内容的问题
- 修复人物八向移动时头部朝向不正确的问题
- 修复 `Invalid assignment of property or key 'modulate' with value of type 'Color' on a base object of type 'Nil'.`
- 修复 `Esc` / `P` 暂停不生效以及 `R` 结算后无法重开的问题

## 当前状态

- Boss 已不再只是单纯压轴单位，而是具备阶段切换和技能循环
- 单局构筑已经能通过升级权重和联动效果形成更明确的打法方向
- 局外成长已经从平铺加点扩展成轻量分支 progression

## 当前仍可继续优化

- UI 视觉还可以继续做更完整的主题化打磨
- Boss 还可以继续补更强的专属演出、独立战场变化和胜利目标
- 局外成长虽然已有分支，但仍可继续扩展更多资源与解锁路线
- 把地图特殊技能卡融合进去
- 修改不要是10分钟的一把游戏
- 生成武器动画

其中地图特殊技能卡、首领目标战、非固定 10 分钟流程和武器动画已在后续 [v0.4.md](D:\javaweb\godot\rouGeDemo\test-rou\version-notes\v0.4.md) 中推进。

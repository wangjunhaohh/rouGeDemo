# v0.2

## 版本目标

把 `v0.1` 的下版本计划真正落地，并处理首版记录中的已知问题。

## 本次新增内容

- 自生成暗黑像素风背景、角色、敌人、投射物与 UI 面板素材
- 自生成 8-bit 风格战斗反馈音效
- 命中粒子、受击反馈、击杀爆散、镜头震动
- 阶段式刷怪节奏与更明显的敌人组合差异
- 精英怪与 Boss
- 结算后可购买的局外成长系统
- 第二轮升级池权重与敌人数值调优

## 已解决的问题

- 不再只有几何图形占位，主场景已替换为像素风素材
- 补上了命中、受击、升级、精英和 Boss 的音画反馈
- 增加了精英怪、Boss 与轻量局外成长
- 升级池加入保底与权重校正，build 倾向更明确

## 当前遗留问题

本节遗留项已在后续 [v0.3.md](D:\javaweb\godot\rouGeDemo\test-rou\version-notes\v0.3.md) 中落实：

- Boss 已补充专属技能与二阶段切换
- 局外成长已扩展为带前置条件的分支
- 人物与相机边界问题已修正
- 人物八向移动的朝向问题已修正
- `modulate on Nil` 报错已修复
- `Esc` / `P` 暂停与 `R` 结算后重开已恢复
- UI 仍可继续主题化，但不再属于阻塞性问题

## 测试结论

- 已使用本机 Godot 4.6 做 headless 加载校验，项目可正常加载主场景
- 素材与音效资源已生成并接入

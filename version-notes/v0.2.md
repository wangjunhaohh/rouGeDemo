# v0.2

## 版本目标

把 `v0.1` 的下版本计划真正落地，并处理首版记录中的已知问题。

## 本次新增内容

- 自生成暗黑像素风背景、角色、敌人、投射物与 UI 面板素材
- 自生成 8-bit 风格战斗反馈音效
- 命中粒子、受击反馈、击杀爆散、镜头震动
- 阶段式刷怪节奏与更明显的敌人组合差异
- 精英怪与 Boss
- 结算后可购买的局外成长系统
- 第二轮升级池权重与敌人数值调优

## 已解决的问题

- 不再只有几何图形占位，主场景已替换为像素风素材
- 补上了命中、受击、升级、精英和 Boss 的音画反馈
- 增加了精英怪、Boss 与轻量局外成长
- 升级池加入保底与权重校正，build 倾向更明确

## 当前遗留问题

- Boss 目前已有压轴出现，但仍缺少更独特的技能机制
- UI 还不是完整成品风格，主要目标仍是服务战斗反馈
- 局外成长深度还不够，后续可以扩展更多分支
- 人物走到屏幕边缘会看到世界外的内容
- 人物八向移动的时候，头部朝向不正确
- 有一个错误bug：Invalid assignment of property or key 'modulate' with value of type 'Color' on a base object of type 'Nil'.

## 测试结论

- 已使用本机 Godot 4.6 做 headless 加载校验，项目可正常加载主场景
- 素材与音效资源已生成并接入

## 下版本计划

- 为 Boss 增加专属技能和阶段切换
- 扩充升级项之间的联动效果
- 把局外成长从轻量加点扩展为更完整的长期 progression
